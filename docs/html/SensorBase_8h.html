Descripción detallada

Clase base abstracta para todos los sensores del sistema IoT.

Define la interfaz común que deben implementar todos los sensores concretos (temperatura, presión, etc.). Declara los métodos virtuales puros procesarLectura() e imprimirInfo() que garantizan el comportamiento polimórfico del sistema.
